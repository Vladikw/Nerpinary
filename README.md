# Веб-приложение "Нерпинарий Иркутска"

Это курсовой проект — веб-приложение для нерпинария города Иркутска. 
Приложение позволяет посетителям просматривать расписание представлений, узнавать информацию о животных и бронировать билеты онлайн.

- Язык программирования: Python 3.10+
- Фреймворк: Django 5.2
- База данных: SQLite (разработка), PostgreSQL (развёртывание)
- HTML, CSS (шаблоны)

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Примените миграции: `python manage.py migrate`
4. Запустите сервер: `python manage.py runserver`
5. Перейдите на http://127.0.0.1:8000/
